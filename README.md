# AWS DynamoDB - Monitoraggio Ambientale

Progetto scolastico sviluppato in gruppo durante il quinto anno (2022), con l’obiettivo di esplorare l’utilizzo di servizi cloud AWS, in particolare DynamoDB, per il monitoraggio e la gestione dei dati ambientali.

## 📖 Descrizione

Il progetto consiste nello sviluppo di un'applicazione in C# per la gestione e il monitoraggio di dati ambientali, utilizzando Amazon DynamoDB come database NoSQL.

L’obiettivo principale è stato comprendere il funzionamento del cloud computing e integrare un sistema backend scalabile per la memorizzazione e l’analisi dei dati.

> ⚠️ Questo progetto è stato sviluppato nel 2022 e potrebbe non essere completamente aggiornato rispetto alle versioni più recenti degli SDK AWS.  

## 🚀 Funzionalità principali

- Connessione a database AWS DynamoDB tramite SDK
- Creazione e gestione di tabelle NoSQL
- Inserimento, lettura e cancellazione di dati
- Monitoraggio e registrazione di informazioni ambientali
- Interfaccia desktop sviluppata in C# (Windows Forms)
- Gestione delle operazioni CRUD su DynamoDB

## 🛠️ Tech Stack

- **Linguaggio:** C#
- **Framework:** .NET (Windows Forms)
- **Database:** AWS DynamoDB
- **SDK:** AWSSDK.DynamoDBv2
- **IDE:** Visual Studio
